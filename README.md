# python_cowmand_line_menu

A simple cowmmand line app i wrote for fun

## Getting Started


Make sure you have python 3, cowsay and fortune installed

```
brew install pyenv
pyenv install 3.3.6
```

```
brew install cowsay
```

```
brew install fortune
```

## Deployment

Fork or clone this repo 

In parent directory run

```
python moonu.py
```
